NS Project. Система управления проектированием

Обеспечивает решение широкого спектра задач по автоматизации процессов управления проектированием. NS Project представляет собой многопользовательскую сетевую информационную систему, построенную на основе клиент-серверной технологии.

Характеристики


Документация

Общие функциональные возможности

Доступна персональная настройка интерфейса под каждого пользователя или преднастроенной роли. Система является инвариантной по отношению к пользователям. Возможности системы задействуются пользователем по мере необходимости, в зависимости от текущих задач.

NS Project является единой средой с единой базой данных, которая логически разделена на функциональные подсистемы. Такой подход позволяет оптимизировать затраты на приобретение и сопровождение лицензий, обеспечивая при этом высокую производительность системы.

Интеграция

Наличие штатного встроенного интерфейса NS Project в NanoCAD, MS Word, MS Outlook. Штатный модуль синхронизации внутренней почты NS Project и корпоративных почтовых серверов позволяет обеспечить автоматическую пересылку уведомлений, напоминаний и подписок на рабочую почту пользователя.

Большой набор API-функций позволяет разрабатывать программы-интерфейсы для интеграции NS Project с внешними приложениями, а также обеспечивает возможность напрямую обращаться к базе данных NS Project из различных внешних приложений.

Защита информации

В NS Project реализован механизм разграничения прав доступа. Система разделения прав доступа является трехуровневой.

Поддерживается дискреционный и мандатный принципы контроля доступа. Механизм защиты от несанкционированного доступа был сертифицирован ФСТЭК по 4 уровню контроля.
